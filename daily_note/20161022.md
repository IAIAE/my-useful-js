#js中语句之间必须用分号隔开
这一点很重要，js引擎会在句末按需添加分号。如果语句之间的分隔不在句末，就需要手动添加分号了。不然会报错。

例如：
```javascript
var a 
=
1 var b=1;
// 这里在var处就会报错。因为前面没有分号，如果敲成
var a
=
1
var b=1;
//就不会报错
```

当然有一种特殊情况：就是语句块，js引擎会自动识别语句块和后一个语句的分割，两者之间不用加分号。
```javascript
loop: while(false){    //while后面接一个语句块，当做一个语句
    consle.info('hello world');
} var b=1;  //这里就不会报错
```
